# NFTMarketPlace
#Requirement:
  + MetaMask
  + NodeJs latest version 
  + NPM
  + Ganache
  + Truffle

#Step to Run:
  1. Clone Github repo to your local computer
  2. Installing the Project Dependancies
    + npm i
  3. Running the App
    + npm start 
  4. Run Ganache
  5. Connecting your metamask to ganache
  6. Upload your pictures to the internet. Ex: https://imgbb.com 
  7. Mint your own NFT
